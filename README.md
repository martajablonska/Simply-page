# Third exercies from Weekly WebDev Challenge

Simply page from Facebook group for learning html, css, js and git. 
My first page from that challenge.

## Technologies
Project is created with:
* HTML 5
* CSS 3 (with flexbox)
* JQuery

I had use parallax scrolling effect (only css), scroll to next sections and hamburger menu (JQuery). 
Page is responsive. 

To made prefixes in css I used https://autoprefixer.github.io/
